## Windows
abrir una terminal
```
ctrl + r
cmd
ok
```

```
C:\Users\harrisod> systeminfo
C:\Users\harrisod> wmic diskdrive get size
```

## Linux

abrir una terminal
```
ctrl + t
```
listar hardware
```
sudo lshw
```
